# TravNL
**TravNL (Project Groenekoek) is een reisplanner zoals de 9292 en NS app gericht op treinreizigers die net wat meer willen.** Je kan meer filters instellen als in de andere reisplanners en meer informatie vinden over specifieke treinen zoals een plattegrond.
Het is een open-source project dat de NS api gebruikt gericht op rail hobbyisten die graag ook een volledige reisplanner alternatief willen als bijvoorbeeld treinposities die ook meer gebruiksvriendelijk is.

## Interface
- Menu aan linkerzijde met 4 opties:
    - Reisplanner 
    - Vertrektijden 
    - Trein nummer zoeken
    - Meer (instellingen)
        - Dark/light mode
        - Thema's
- Geavanceerde reisplanner
    - Algemene filters zoals o.a datum en tijd
    - Extra looptijd instellen
    - Specifieke OV types
    - Extra filters
- Geavanceerde trein info (bij tikken op trein of in treinnr zoeker)
    - Trein informatie 
        - Trein type
        - Aantal bakken
        - Verwachtte drukte 
        - Plattegrond met locatie van
            - 1e klas
            - Fiets opstelplaats en beschikbaarheid
            - Rolstoelplaatsen
            - Toiletten
        - Aankomst en vertrektijden van stations en passerende stations (waar de trein niet stopt)
